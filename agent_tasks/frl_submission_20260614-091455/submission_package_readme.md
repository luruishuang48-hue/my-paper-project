# FRL Submission Package

This folder contains a compact main paper and a longer online appendix.

- `frl_submission_main.tex` is the main manuscript.
- `frl_online_appendix.tex` is the online appendix.
- `highlights.txt` contains five short highlights.
- `data_availability_statement.md` contains the data availability language.
- `cover_letter_draft.md` contains a journal cover letter draft.
- `appendix_architecture.md` records the appendix design review.
- `submission_risk_review.md` records remaining interpretation and submission risks.

The main paper deliberately foregrounds the metric comparison and the closed-source AA Intelligence result. Broader heterogeneity, full specification-curve evidence, overlap diagnostics, and exploratory relationship results stay in the appendix.

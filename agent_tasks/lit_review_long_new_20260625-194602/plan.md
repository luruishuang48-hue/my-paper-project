# Multi-Agent Execution Plan

任务：为 `Tex/long_new.tex` 补写并修订文献综述，重点覆盖科技产品发布的事件研究、AI 与金融市场，并补充相关领域经典和 2020 年后的英文文献。

时间戳：20260625-194602（Asia/Shanghai）

## 一、规划阶段

1. 读取 `Tex/long_new.tex` 当前结构，确认第二节已有文献综述的占位和引用方式。
2. 建立研究资料目录，按主题拆分文献检索。
3. 并行调研三个相互独立的文献块。
   - A 组负责科技产品发布、IT 投资、创新公告、专利和产品预告的事件研究。
   - B 组负责 AI、ChatGPT、生成式 AI 与金融市场、公司估值、劳动暴露和资产定价。
   - C 组负责供应链外溢、竞争效应、互补品/替代品、开源经济学和平台生态。
4. 主代理同步处理目标文稿结构、引用格式、中文表达和假设段落衔接。

## 二、实施阶段

1. 各子代理将完整调研结果保存到本目录下的 `research_*.md` 文件，只回传简短状态。
2. 主代理整合文献，形成可直接进入论文的中文综述段落。
3. 修改 `Tex/long_new.tex` 的第二节，必要时同步补充参考文献环境。
4. 保持论文已有证据分级逻辑，避免把尚未识别为因果的关系写成强因果。

## 三、审阅阶段

1. 检查文献覆盖是否满足三类要求。
   - 领域经典。
   - 2020 年以后研究。
   - 英文文献为主，中文文献不强求。
2. 检查引用是否与正文论断匹配。
3. 检查中文是否自然，减少冒号使用，避免翻译腔和空泛表述。
4. 检查 LaTeX 是否能编译，若失败则定位并修复语法问题。

## 四、修订阶段

1. 根据审阅结果压缩重复内容，强化与本文贡献的连接。
2. 修正引用、参考文献条目和 LaTeX 语法。
3. 产出最终说明，列出改动文件、检索记录和验证结果。

## 当前成功标准

1. `Tex/long_new.tex` 的文献综述不再有待补充占位。
2. 第二节能清楚承接本文三个假设。
3. 参考文献包含核心经典和 2020 年后代表性研究。
4. 文稿能通过一次 LaTeX 编译或明确说明无法编译的外部原因。
